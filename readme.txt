== Description ==

DesignContest Feed - display active contests feed from www.designcontest.com at your website in 2 clicks.

== Installation ==

1. Upload `dcrss.php` and `dcstyle.css` to the `/wp-content/plugins/` directory of your WordPress installation
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The DesignContest Feed widget can now be configured and used from the Appearance -> Widgets menu
